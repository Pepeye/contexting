# Bun Development Utilities

Essential Bun development commands and utilities.

## Quick Commands

```bash
# Development server
bun run dev

# Package management
bun install
bun add <package>
bun remove <package>

# Testing
bun test
bun test --watch

# Building
bun run build

# Type checking
bun run typecheck
```

## Common Patterns

- Use TypeScript for type safety
- Leverage Bun's native APIs
- Prefer async/await over Promises
- Use modern JavaScript features
